# Newsletter Form - Frontend Mentor

A newsletter subscription form project developed as a **Frontend Mentor** challenge. This project is an interactive application with email validation and visual feedback for users.

## Project Description

This is a responsive newsletter form that allows users to:
- Subscribe to a newsletter using their email address
- Receive real-time validation of the entered email
- View a confirmation screen after a successful subscription
- Optimized user experience with modern styles

## Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Styles and responsive design
- **JavaScript (Vanilla)** - Application logic
- **Vite** - Build tool and dev server
- **Custom Vite Plugin** - Custom plugin for HTML components

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/FelipeAlvarad63/Newsletter-form-Frontend-Mentor.git

# Navigate to the directory
cd Newsletter-form-Frontend-Mentor

# Install dependencies
npm install
```

### Available Scripts

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview the build
npm run preview
```
